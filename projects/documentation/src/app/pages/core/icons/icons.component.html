Icons

Pastanaga 3 includes heroicons as a sprite of svg icons and the component pa-icon allowing to display them.

The usage of this component is very simple, it only has one name property allowing to chose which icon to be displayed:

{{ codeExample }}

You can then style the icon using any tailwindcss class on the component. Their color is based on the text color:

{{ styleExample }}

By default, width and height are set to w-6 h-6 (so 24px) but you can override them by prividing any size class:

{{ sizeExample }}

Generating icon sprite

Pastanaga includes heroicons outlined icons. They are stored in ngx-pastanaga/glyphs folder. Pastanaga also provides a script allowing to turn icons stored in this folder into a single svg sprite generated in ngx-pastanaga/src/assets folder (and copied into documentation/public/assets folder to be used in the documentation app).

Whenever you update any svg in the glyphs folder, you must also update the svg sprite by executing the following command from the workspace root:

./scripts/update-glyphs.sh
