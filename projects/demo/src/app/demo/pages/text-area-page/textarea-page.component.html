Textarea

pa-textarea wraps an html textarea element and apply pastanaga's form field style.

As other pastanaga's form fields, pa-textarea supports 4 modes internally: standalone, model, formControl and formControlName, in order to be used in various context.

pa-textarea also supports internal and external validation (required and maxlength).

pa-textarea elements can be made to automatically resize. The resizable property of the field can also be disabled.

Inputs

autoHeight
allow textarea to expand with user entries or according to currentValue: boolean (default: false)
resizable
allow user to resize the textarea: boolean (default: true)
rows
textarea's rows attribute: number (default: 1)
maxRows number
defines the max count of rows when used in combination with autoHeight. Unused when a maxHeight is defined (optional)
maxHeight number
defines the maxHeight of the textarea (optional)
