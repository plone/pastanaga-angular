paFormControl accepts internal validators. This allow the directive to programmatically set a custom errorMessage using the errorMessage input.

Important: validation will only be evaluated when the formControl is dirty.

{{errorMessageCode}}
formControl errors: {{errors | json}}

When used in combination with ngModel or formControl directives, paFormControl internal validators are merged with parent's validators.
