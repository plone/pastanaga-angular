pa-form-control exposes form control inner state and makes touched / pristine / dirty / pending / valid values accessible.
NB: ngControlStatus classes will only be available when used with ngModel and formControls, but the formControl's state can be used to create our own style.

  • .ng-untouched
  • .ng-touched
  • .ng-pristine
  • .ng-dirty
  • .ng-valid
  • .ng-invalid
  • .ng-pending
