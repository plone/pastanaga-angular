paFormControl can be marked as disabled or readonly .

Important : disabled can not be used in combination with FormControl or FormControlName directive. 

{{code}}
standalone formControl state: disabled({{ controlDisabled }}), readonly({{ controlReadonly }})
ngModel formControl state: disabled({{ modelDisabled }}), readonly({{ modelReadonly }})


paFormControl exposes form control inner state and makes touched / pristine / dirty / pending / valid values accessible.
NB: ngControlStatus classes will only be available when used with ngModel and FormControl , but the formControl's state can be used to create our own style.

  • .ng-untouched
  • .ng-touched
  • .ng-pristine
  • .ng-dirty
  • .ng-valid
  • .ng-invalid
  • .ng-pending
