Name attribute is used whenever provided, if not, a default name is built based on the form-control id.
However, used inside forms, the name attribute will follow angular api:

  • name attribute is mandatory in template driven forms
  • name must be equal to formControlName in reactive forms

Standalone

{{standaloneCode}}
provided
formControl name: {{standaloneName1}}
generated
formControl name: {{standaloneName2}}


Template Driven Form

{{templateDrivenCode}}
formControl name: {{firstName}}
formControl name: {{secondName}}
second value
formControl name: {{thirdName}}
formControl name: {{fourthName}}
fourth value

{{reactiveCode}}

Provided names will always be ignored
