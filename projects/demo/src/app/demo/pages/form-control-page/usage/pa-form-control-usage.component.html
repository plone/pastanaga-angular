Inputs

id
id to be used by the directive (optional)
name
name to use for the form field (optional)
value
sets the value to the formControl (should not be used in combination with ngModel nor formControl directives)
readonly
sets the formControl to readonly (value can only be updated by parent component) (default: false)
disabled
disable the form control (default: false)
errorMessage
manually set the control in invalid state with the given error message (optional)

Outputs

valueChange
emits the value when it changes
statusChange
emits the status of the form control value when it changes (valid / invalid)
