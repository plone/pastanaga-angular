Ellipsis tooltip

The directive paEllipsisTooltip is adding style on its parent to display an ellipsis when there is not enough space to display all the text.

Whenever the element has an ellipsis, this directive automatically adds a system tooltip displaying the full text content on hover.

Short text

No tooltip

Text content long enough to display an ellipsis and a tooltip

Add paEllipsisTooltip directive wherever you want to display an ellipsis when the content is too long for the container.

Style responsible for the ellipsis will be added automatically, but you need to manage the container size.

Inputs

paEllipsisContent
By default, the tooltip contains the same text as its parent element. paEllipsisContent allows to change the content of the tooltip displayed.
noEllipsis
Allows to use paEllipsisTooltip on an element and optionally disable it. (False by default)

Outputs

hasEllipsis
Emit a boolean indicating if the content has an ellipsis or not. Event emitted on AfterViewInit and then anytime content changed (when paEllipsisContent Input is applied). (Event emitted from updateEllipsisTooltip )
updateEllipsisTooltip
this method from paEllipsisTooltip is public and can be called externally, specially when the content is dynamic and the container needs to react to the presence of an ellipsis.
