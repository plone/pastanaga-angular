Pastanaga provides scss classes for building tables based on grid layout. Those classes are available in pastanaga-angular/src/styles/_tables.scss .

Pastanaga provides also some components and directives for building such tables more easily.

For basic tables:
  • pa-table : Table wrapper component, required for all tables. It can contain pa-table-header , pa-table-sortable-header , pa-table-row , and pa-table-row-header
  • pa-table-header : directive simply wrapping table header cells. It can contain pa-table-cell and pa-table-cell-menu
  • pa-table-row-header : it renders as a full width row, displaying a label. Free content.
  • pa-table-row : it can contain pa-table-cell , pa-table-cell-menu and table-lead-cell-multi-line
  • pa-table-cell : free content.
  • pa-table-cell-menu : extends pa-table-cell component, and renders as a cell with content centered. This kind of cell is meant to wrap a pa-button , on which specific style is applied on hover and when active.
For sortable tables:
  • pa-table-sortable-header : easy way to manage sorting on a table.
  • pa-table-sortable-header-cell : renders a sortable header cell with a sorting indicator.
For mobile display:

On mobile, there is usually not enough space to display a table with several columns. So Pastanaga provides a component pa-table-lead-cell-multi-line and some directives to display the most important information of a row in one cell.

Even though it was designed with mobile tables in mind, this component can be used as a cell in any table row.

  • pa-table-lead-cell-multi-line : component allowing to display an image on the left, followed by a title and a description.
  • pa-table-lead-image : directive wrapping the image to be displayed
  • pa-table-lead-title : directive wrapping the title to be displayed
  • pa-table-lead-description : directive wrapping the description to be displayed
