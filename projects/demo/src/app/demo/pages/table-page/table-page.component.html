pa-table

It can contain pa-table-header, pa-table-row, and pa-table-row-header

columns
Define the grid-template-columns CSS attribute value. (optional, default is auto)
noHeader
If true, table header is not displayed. (optional)

pa-table-header

It can contain pa-table-cell and pa-table-cell-menu

pa-table-row

It can contain pa-table-cell and pa-table-cell-menu

clickable
If true, hovering the row will display a pointer cursor. (optional)
pa-selected
Applying the class .pa-selected to a row change its background color to be $color-background-table-row-selected.

pa-table-row-header

It renders as a full width row, displaying a label. Free content.

pa-table-cell

It renders as a cell. Free content.

header
If true, cell is rendered as a header cell. (optional)

pa-table-sortable-header-cell

It renders as a sortable header cell with a sorting indicator.

active
If true, display the sorting indicator icon, otherwise it will display a chevron down icon on hover. False by default
isDescending
Set the sorting indicator to sorted-descending icon. By default, sorting indicator is sorted-ascending
sort
Event triggered when clicking on the cell.

pa-table-cell-menu

It renders as a cell with centered aligned content and it must contain a pa-button.

The button is visible only when its parent row is hovered (except for header cells).

header
If true, cell is rendered as a header cell and menu button is always visible. (optional)
