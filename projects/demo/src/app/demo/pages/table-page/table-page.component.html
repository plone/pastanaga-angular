pa-table

It can contain pa-table-header, pa-table-sortable-header, pa-table-row, and pa-table-row-header

columns
Define the grid-template-columns CSS attribute value. (optional, default is auto)
noAutoColumnStyle
By default, the table set grid-template-columns: auto when columns property is not filled. To prevent this behavior, set noAutoColumnStyle property on your table. (optional, false by default)
noHeader
If true, table header is not displayed. (optional)

pa-table-header

It can contain pa-table-cell and pa-table-cell-menu

pa-table-sortable-header

Easy way to manage sorting on a table. On mobile, only the sorted column is displayed and clicking on it display a dropdown with all the sortable columns.

cells
List of HeaderCell to display.
menuColumn
Add an empty menu column when set (optional)
sort
Emit sorted HeaderCell when sorting changes

pa-table-row

It can contain pa-table-cell and pa-table-cell-menu. Applying the class .pa-selected to a row change its background color to be $color-background-table-row-selected.

clickable
If true, hovering the row will display a pointer cursor. (optional)
disabled
Disable the row by applying an opacity mask and preventing pointer-events. (default: false)

pa-table-row-header

It renders as a full width row, displaying a label. Free content.

pa-table-cell

It renders as a cell. Free content.

header
If true, cell is rendered as a header cell. (optional)
disabled
Disable the cell by applying an opacity mask and preventing pointer-events. (default: false)
clickable
If true, hovering the cell will display a pointer cursor. (optional)
noWrap
Prevent the cell to wrap its content. You can then apply the class pa-ellipsis to your cell content. (Default: false)

pa-table-sortable-header-cell

It renders as a sortable header cell with a sorting indicator.

active
If true, display the sorting indicator icon, otherwise it will display a chevron down icon on hover. (Default: false)
isDescending
Set the sorting indicator to arrow-up icon. By default, sorting indicator is arrow-down
sort
Event triggered when clicking on the cell.

pa-table-cell-menu

It renders as a cell with centered aligned content and it must contain a pa-button.

header
If true, cell is rendered as a header cell and menu button is always visible. (optional)
