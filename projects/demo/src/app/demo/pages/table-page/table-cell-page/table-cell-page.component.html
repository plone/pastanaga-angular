Table cell

Inputs

center
If true, cell's content is horizontally centered. (default: false)
clickable
If true, hovering the cell will display a pointer cursor. (default: false)
disabled
Disable the row by applying an opacity mask and preventing pointer-events. Disabled cell is not focusable. (default: false)
header
If true, apply the header cell style. Apply basic cell style otherwise. (default: false)
noWrap
Prevent the cell to wrap its content (with white-space: nowrap). You can then apply the class pa-ellipsis to your cell content. (Default: false)
