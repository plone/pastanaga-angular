Confirmation dialog

Pastanaga provides a full system to manage modals and dialogs which is documented here.

To simplify the creation of commonly used confirmation dialog, Pastanaga also provide a method openConfirm as part of the modalService to directly open a confirmation dialog.

This method takes an attribute ConfirmationData as unique parameter to configure the confirmation dialog.

 Open a confirmation Open a destructive confirmation

ConfirmationData

title
Title to be displayed in the confirmation dialog. This is the only mandatory property.
description
Description to be displayed below the title. Optional
isDestructive
When true, the confirmation button will be of kind destructive. Default: false
cancelLabel
Label to display on the cancel button. Default: pastanaga.cancel
confirmLabel
Label to display on the confirm button. Default: pastanaga.confirm
