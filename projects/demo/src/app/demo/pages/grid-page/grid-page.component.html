The grid is 12-based, but grid with less columns can be created, example: class="grid-4".

.grid-offset allows to offset by a given number of columns, example: class="col-4 grid-offset"

Columns can be hidden/visible depending on the viewport size, examples:class="col-4 small-hidden tiny-hidden", class="col-4 large-visible".

Horizontal alignment is performed with .justify-center and .justify-right.

Vertical alignment is performed with .align-center and .align-bottom.
