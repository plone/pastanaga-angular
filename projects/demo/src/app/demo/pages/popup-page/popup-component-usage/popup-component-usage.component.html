Inputs

companionElement
When clicking on the companion element – which is outside the popup and must be an HTML element – the popup won’t close. It offers external interactions (like filtering the popup content for example). (optional)
When companionElement is an angular component, you must provide its nativeElement, eg. [companionElement]="somePaInput.element.nativeElement"
dontAdjustPosition
Keep default popup position even if there is not enough space to display it entirely. (default: false)
id
Popup's id (optional, an id will be generated if none is provided)
keepOthersOpen
Control if other popups should be closed when current one is opened. (default: false)
stayVisible
Keep the popup visible when set to true (default: false)

Outputs

onClose
Event emitted whenever the popup is closed
onOpen
Event emitted whenever the popup is opened
