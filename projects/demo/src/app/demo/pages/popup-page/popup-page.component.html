Popup

Pastanaga provides a component and a directive allowing to give a popup behaviour to any component.

  • PopupComponent: Component to be extended by any component needing a popup behaviour.
  • PopupDirective: Directive allowing to open a popup by clicking on the element it belongs to

Configuration options

Open popup on right

PopupComponent

Inputs

id
Popup's id (optional, an id will be generated if none is provided)
companionElement
When clicking on the companion element – which is outside the popup – the popup won’t close. It offers external interactions (like filtering the popup content for example)(optional)
isAlwaysOn
Keep the popup visible when set to true (default: false)

Outputs

onClose
Event emitted whenever the popup is closed

PopupDirective

Inputs

paPopup
Popup to open when clicking on the element carrying the directive
popupPosition
Position style to be used on the popup (optional: by default the popup is positioned on top of the element carrying the directive)
popupOnRight
Set the popup on the right of the element carrying the directive (default: false)
