Popup

Pastanaga provides a component and a directive allowing to give a popup behaviour to any component.

  • PopupComponent: Component to be extended by any component needing a popup behaviour.
  • PopupDirective: Directive allowing to open a popup by clicking on the element it belongs to

PopupDirective

Inputs

openOnly
Don't close popup when clicking on the element carrying the directive while the popup is visible. (default: false)
paPopup
Popup to open when clicking on the element carrying the directive
popupDisabled
Don't open popup when clicking on the element carrying the directive
popupPosition
Position style to be used on the popup (optional: by default the popup is positioned on top of the element carrying the directive)
popupOnRight
Set the popup on the right of the element carrying the directive (default: false)
popupOnTop
Set the popup on the top of the element carrying the directive (default: false)
sameWidth
Set the popup width equals to the element carrying the directive (default: false)
 
{{example}}
To open a popup programmatically, in your template add a reference to your popup directive: 
{{openProgrammaticallyTemplate}}
Then in your component, you can call toggle method of your directive to open/close the popup: 
{{openProgrammaticallyComponent}}
