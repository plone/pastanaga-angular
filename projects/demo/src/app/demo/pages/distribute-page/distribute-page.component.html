Distribute

Distribute scss variables define the space between blocks of content: it is meant to be used as the separation between containers inside the main container.

Can be used for both vertical and horizontal spaces.
All viewports
xsmall (16px – 2 rhythm)
small (24px – 3 rhythm)
Viewport small
medium (32px – 4 rhythm)
large (40px – 5 rhythm)
Viewport medium and large
medium (48px – 6 rhythm)
large (80px – 10 rhythm)
 Scss variables: 

$distribute-xsmall-viewport-all // can be used in any media query
$distribute-small-viewport-all  // can be used in any media query

$distribute-medium-viewport-small // to be used on small devices only
$distribute-large-viewport-small  // to be used on small devices only

$distribute-medium-viewport-medium // to be used on medium and bigger viewports
$distribute-large-viewport-medium  // to be used on medium and bigger viewports
