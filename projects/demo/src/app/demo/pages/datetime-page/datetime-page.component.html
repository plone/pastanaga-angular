Date/time Date/time formatting component, based on app locale. Human Human, date only Numerical Numerical, date only
datetime
String. The date/time to display, in ISO format.
format
String (optional, default human )
  • human
    • format few seconds/minutes ago as relative time like 'Just now' or '10 minutes ago'
    • format today/yesterday as human date like 'Today, 8:13AM'
    • format date before yesterday like 'Dec 31, 2017' or just 'Oct 12' if in the current year
  • numerical
    • format date as numerical date like '12/31/2017'
dateOnly
Boolean. Do not display time (optional, default false )
displaySeconds
Boolean. Display seconds (optional, default false )
