paNativeTextField

An extension of paFormControl adding functionalities for native html fields

paNativeTextField is designed to be a base class of custom form controls.

It manages a ViewChild input or textarea html element and allows controls on:

  • placeholder
  • maxLength html attribute
  • enable / disable browser autofill
  • provide hint and error messages to the user
  • focus the input programmatically
  • fixes iOS issues with caret positioning
  • gives access keyup event
  • gives access keyup.enter event
  • gives access to focus event
  • gives access to blur event
  • allow to 'sanitize' user input, removing html tags
