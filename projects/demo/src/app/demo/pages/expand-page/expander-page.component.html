Expander

Expander component allows to expand/collapse a section by clicking on its header or on the chevron button next to it.

By default, expander is displayed like in a list: the toggle button being on the left. The card option allows to display a card expander with the toggle button on the right.

PaExpanderModule provide different directives to be used with pa-expander:

  • pa-expander-header: its content is displayed next to the chevron button, clicking on it toggles the expander
  • pa-expander-body: its content is displayed below expander's header. Visible only when expanded.
  • pa-expander-header-side-block: its content is displayed next to the pa-expander-header. Clicking on it doesn't toggle the expander.

Inputs

contentLoaded
Any value marking expander's content as loaded or updated so the expand can update its body height.
card
Card option allows to change the expander look and feel by displaying toggle button on the right and any pa-expander-header-side-block on the left. When this option is set, the card expander is collapsed by default and can only be expanded by the expander button
disabled
In case we want the expander to be disabled
