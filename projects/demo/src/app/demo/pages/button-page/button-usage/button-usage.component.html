Inputs

disabled
Set disabled state
icon
Icon name to be used. Button text will be hidden but accessible by screen reader.
iconSize
Set the size of the icon: small | medium | large. (default: medium)
iconAndText
Display icon and text together (default: false)
kind
primary | secondary | destructive | inverted (default: secondary)
size
small | medium | large (default: medium). By default, icons in small buttons have a medium size, but you can override this behaviour by setting iconSize attribute.
type
button | submit | reset (default: button)
aspect
solid | basic (default: solid)
