Inputs

disabled
Set disabled state
icon
Icon name to be used. Button text will be hidden but accessible by screen reader.
iconAndText
Display icon and text together (default: false)
id
Button's id (optional, an id will be generated if none is provided)
kind
primary | secondary | critical (default: secondary)
size
xsmall | small | medium | large (default: medium)
type
button | submit | reset (default: button)
weight
accent | regular | dim (default: accent)
