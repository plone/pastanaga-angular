Pastanaga provides a button component with:

  • 4 different kinds : primary , secondary , destructive and inverted
  • 2 different aspects : solid and basic
  • 3 different sizes : small , medium and large

The text wrapped by pa-button tag is used as aria-label for accessibility purpose. As a good practice, you should always set a text content for your buttons, even for icon buttons: the text won't be visible on screen but it will be accessible for screen readers.

There is no specific component for button links because you can wrap a link directly in the pa-button tag: 

{{buttonLinkCode}}

If the routerLink directive is only on the link, the user will have to click on the link text to be redirected. To make sure the user can click on any part of the button to be redirected, you should add the routerLink directive on the button itself. Putting the routerLink on the {{ paButtonTag }} only is also working, but keeping the link tag can be good for semantic and accessibility reasons.

Links wrapped by pa-button have their style overridden so their color is inherited from the button style, and they have no decoration on hover.
