Input

pa-input wraps an html input element and apply pastanaga's form field style.

As other pastanaga's form fields, pa-input supports 4 modes internally: standalone, model, formControl and formControlName, in order to be used in various context.

pa-input also supports internal and external validation.

Set autocomplete attribute to off (default: false)
acceptHtmlTags
Allows to render/enter HTML tags in the field. Enabling this can have an impact on the security of your app, to be use with caution. (default: false)
autocapitalize
Allow to define the HTML5 autocapitalize attribute value. (no default)
Note: it must be set to off to disable the autocapitalization applied by some smartphones.
hasFocus
triggers the focus on the input field (optional)
help
A hint to display, helping the user when filling a form field (optional)
showAllErrors
Display all errors or only the first error encountered (errors being sorted by key) (default: true)
errorMessages
Provides default messages to display for the possible error keys when none is present in the errors object. If a message (string) is provided for a given key, it will be displayed rather than the message provided in errorMessages.

Outputs

blurring
Emits current value on blur
enter
Emits KeyboardEvent and string value when pressing enter
focusing
Emits FocusEvent on focus
valueChange
Emits value as soon as it changes. Only available when input is in standalone mode.
keyUp
Emits string value on key up event
