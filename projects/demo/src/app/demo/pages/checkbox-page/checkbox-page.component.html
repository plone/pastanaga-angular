Checkboxes

pa-checkbox wraps an input element of type checkbox and applies pastanaga's form field style.

As other pastanaga's form fields, pa-checkbox supports 4 modes internally: standalone, model, formControl and formControlName, in order to be used in various context.

pa-checkbox uses pa-form-field-hint to display error messages.

Important note: readonly is not applicable to input of type checkbox so the input inherited from PaFormControlDirective will have no effect.
