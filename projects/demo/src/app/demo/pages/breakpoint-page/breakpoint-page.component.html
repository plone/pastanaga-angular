Breakpoint observer

Pastanaga theme provides some breakpoints variables so you can easily manage your responsive layouts. As layout can not always be managed entirely through css, Pastanaga provides also a BreakpointObserver service allowing to watch viewport current min/max size.

Types

ViewportSize
Enum containing viewport sizes supported by Pastanaga (corresponding to the ones defined in css)
ViewportMode
'desktop' | 'tablet' | 'mobile'

BreakpointObserver

currentMinSize
Observable returning ViewportSize corresponding to the current min-width from css media query
currentMaxSize
Observable returning ViewportSize corresponding to the current max-width from css media query
currentMode
Observable returning ViewportMode based on currentMinSize
