Breakpoint

Pastanaga-angular provides some scss variables for breakpoints so you can easily manage your responsive layouts. Breakpoints are defined as:

  • Small: 0px to 599px
  • Medium: 600px to 1023px
  • Large: 1024px to 1439px
  • XLarge: 1440px and more

As layout can not always be managed entirely through css, Pastanaga provides also a BreakpointObserver service allowing to watch viewport current min/max size.

Scss

$size-viewport-medium-min
Token to be used in min-width media queries for medium screens
$size-viewport-large-min
Token to be used in min-width media queries for large screens
$size-viewport-xlarge-min
Token to be used in min-width media queries for xlarge screens

Typescript

Types

ViewportSize
Enum containing viewport sizes supported by Pastanaga (corresponding to the ones defined in css)
ViewportMode
'desktop' | 'tablet' | 'mobile'

BreakpointObserver

currentMinSize
Observable returning ViewportSize corresponding to the current min-width from css media query
currentMaxSize
Observable returning ViewportSize corresponding to the current max-width from css media query
currentMode
Observable returning ViewportMode based on currentMinSize
