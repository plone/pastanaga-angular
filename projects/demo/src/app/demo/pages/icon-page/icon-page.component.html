Icons

Pastanaga provides a component pa-icon allowing to display an icon from the glyph-sprite by using the icon name. pa-icon can also be used to display any image by providing their path.

Icons are inheriting their colors from the text color. Overriding an icon color is as simple as adding a css class on the pa-icon changing the text color.

In your application, you can either use your own glyph sprite or you can use Pastanaga one. If you're using your own sprite, just make sure to include glyphs with the following names in your sprite as some Pastanaga components are using them:

  • calendar: used in date-picker component
  • chevron-down: used in chip-expandable, select, date-picker, expander and tabs components
  • chevron-left: used in date-picker component
  • chevron-right: used in date-picker component
  • cross: used in chip-closeable, modal-advanced and side-nav components
  • more-horizontal: used in avatar-pile component

You must fill one of the following properties: name or path .

name
Icon id to be found in svg sprite. Will be resolved as {{ nameResolvedExample }} . (optional)
path
Path of the icon in asset folder. (optional)
size
Size to be applied: small | medium | large. (default: medium)
color
Color to apply to the icon (works only with svg icons). (optional, can be useful if you need to apply a color coming from the backend)
background
Background color to apply to the icon (optional)
