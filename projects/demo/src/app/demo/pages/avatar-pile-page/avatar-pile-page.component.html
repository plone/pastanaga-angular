Avatar pile

Display a pile of small avatars.

Avatar pile has a maximum of 3 avatars before showing a horizontal ellipsis button.

Avatar pile doesn't sort the avatars: they are displayed in the exact same order as they are provided.

By default, clicking on the button display a dropdown with the list of all avatars. You can setup your own behavior by setting customButton Input and listening to clickOnMore event.

Default behavior: button visible when more than 3 avatars in the pile

Optional behavior: button always visible

Custom button behavior

Inputs

avatars
List of avatars to display as a pile. A maximum of three avatars will be shown in the provided order (no internal sorting).
buttonAlwaysVisible
When set, the button is visible whatever the number of avatars in the list. (Default: false)
customButton
When set, clicking on the button won't open a dropdown with the list of users. (Default: false)

Outputs

clickOnMore
Event emitted when clicking on the button
