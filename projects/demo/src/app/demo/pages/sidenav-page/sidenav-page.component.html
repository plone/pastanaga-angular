Sidenav

pa-side-nav component allows to add a side navigation to your app.

The side navigation is always visible on desktop, while it is hidden and displayed only on user action such as clicking on a menu button on mobile and tablet.

pa-side-nav

It can contain pa-side-nav-header, pa-side-nav-content, and pa-side-nav-footer.

Visible
Manage the visibility of the component.
Mode
mobile | tablet | desktop

pa-side-nav-header

By default, the sidenav display only a close button on mobile/tablet. You can provide your own header within pa-side-nav-header.

pa-side-nav-content

Container for pa-side-nav-item.

pa-side-nav-item

header
If true, it is rendered as a header.
label
Text to display.
active
If true, active styles will be applied.
