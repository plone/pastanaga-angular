Sidenav

pa-side-nav is used to add menu content to your app

pa-side-nav

It can contain pa-side-nav-header, pa-side-nav-content, and pa-side-nav-footer

Visible
Manage the visibility of the component.
Mode
mobile | tablet | desktop

pa-side-nav-header

It works as a header for the sidenav, the content will be aligned to the top.

pa-side-nav-content

It'll contain pa-side-nav-item

pa-side-nav-item

Header
If true, it is rendered as a header.
Label
Text to display.
Active
If true, active styles will be applied.

pa-side-nav-footer

It works as a footer for the sidenav, the content will be aligned to the bottom.
