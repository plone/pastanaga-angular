Chips

There are three kinds of chip: closeable chips , selectionable chips and expandable chips .

All kinds have the same look and feel and can display an avatar and/or an icon on the left.

  • pa-chip-closeable display a close button on the right of the chip. The attribute noCloseButton allows to remove the close button.
  • pa-chip-selectionable don't have any button but triggers a select event on click and applies specific style when selected
  • pa-chip-expandable display a chevron button on the right. It triggers an expanded event on click and applies specific and toggle the chevron to be up or down.

Common inputs

ariaRole
Defines the role attribute of the chip. (listitem by default)
avatar
AvatarModel configuring the avatar to display. (optional)
backgroundColor
Hexadecimal color used to override the default background color. This colors is not applied on disabled chip. (optional)
borderColor
Hexadecimal color used to override the default border color. This colors is not applied on disabled chip. (optional)
disabled
Disable the chip. (optional)
icon
IconModel configuring the icon to display. (optional)
textColor
Hexadecimal color used to override the default text color. This colors is not applied on disabled chip. (optional)
value
A value held by the chip. (optional)

pa-chip-closeable

Inputs

noCloseButton
Removes the close button from the chip. (optional)

Outputs

closed
Emits when the closed button is clicked. Provides the event and the optional value of it.

pa-chip-selectionable

Inputs

selected
Boolean applying selected style to the chip

Outputs

select
Event trigger when clicking on the chip. Triggered only if not disabled

pa-chip-expandable

expanded
Event triggered when clicking on the chip. It contains the mouse or keyboard event itself as well as the chip value. Triggered only if not disabled
