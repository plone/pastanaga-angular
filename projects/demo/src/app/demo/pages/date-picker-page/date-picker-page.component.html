Date Picker

pa-date-picker wraps an html input element and applies pastanaga's form field style.

As other pastanaga's form fields, pa-date-picker supports 4 modes internally: standalone, model, formControl and formControlName, in order to be used in various context.

pa-date-picker also supports internal and external validation.
