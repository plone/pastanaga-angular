Inputs

active
Adds an outline-like border to the component to simulate a focused or active button
alternateText
The text used to fill alt property of the image. When Avatar is displayed as an Icon, and the size is large, this text can also provide a label to the icon. (optional)
autoBackground
Let component choose initial's background color according to userName or userId. false by default
backgroundColor
Hexadecimal value to be used as background color when no image is provided. If autoBackground is true, backgroundColor overrides the generated one. (optional)
icon
Icon name to be used. When an image or a userName is provided, the icon will be displayed on the bottom right corner of the avatar. Otherwise, the icon will be displayed as an avatar itself. (optional)
iconBackgroundColor
Hexadecimal color of icon's background. (optional)
iconColor
Hexadecimal color of the icon. (optional)
image
Provides to be displayed as avatar. Expect an observable of blob. Once an image is provided, generated initials and backgroundColor are not displayed anymore. (optional)
large
Display large avatar false by default
userId
Used to generate a background color when no image or backgroundColor is provided. (optional)
userName
Used to generate displayed initials and a background color when no image or backgroundColor is provided. (optional)
