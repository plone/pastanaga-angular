Avatar You can use an Avatar to represent a user or a group. Avatars with no image can have an auto assigned background color.
Large
Medium
Small

Inputs

autoBackground
Assign background color automatically according to userName or userId. (false by default)
image
Observable of blob to be displayed as avatar. Once the image is set, generated initials and backgroundColor are not displayed anymore. (optional)
imageSrc
Base64 string to be displayed as avatar. Once the image is set, generated initials and backgroundColor are not displayed anymore. (optional)
size
Avatar's size. Can be small | medium | large. (medium by default)
userId
Used to generate a background color when no image is provided and autoBackground is true. (optional)
userName
Used to generate displayed initials and a background color when no image is provided and autoBackground is true. (optional)
