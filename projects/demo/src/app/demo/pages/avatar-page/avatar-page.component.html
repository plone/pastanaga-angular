Inputs

userId
Used to generate a background color when no image or backgroundColor is provided.
userName
Used to generate Displayed initials and a background color when no image or backgroundColor is provided.
backgroundColor
Specifies the hexa background color to use when no image is provided. Generated backgroundColor will then be override.
image
Observable of blob. Provides the image to the component. Once an image is provided, generated initials and backgroundColor are not displayed.
large
Toggles avatar display size.
autoBackground
Let component choose initial's background color according ti userName or userId.
icon
Icon name to be used. When an image or a userName is provided, the icon will be displayed on the bottom right corner of the avatar. Otherwise, the icon will be displayed as an avatar himself.
iconColor
Hexa color of the icon.
iconBackgroundColor
Hexa color of icon's background.
alternateText
The text used to fill alt property of the image. When Avatar is displayed as an Icon, and the size is large, this text can also provide a label to the icon.
