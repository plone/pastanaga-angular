Tabs

Pastanaga tabs organize content into separate views where only one view can be visible at a time.

Each tab's label is shown in the tab header and the active tab's label is highlighted with an animated primary color underline by default.

pa-tabs

noSlider
When true, no slider is displayed under the tab list. (default: false)
notFullWidth
When true, the tab item width depends on their content. (default: false, meaning tab items take all the available space with flex grow)
