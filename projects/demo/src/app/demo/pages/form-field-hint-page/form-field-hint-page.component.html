Form field hint A wrapper used to describe a form field with hint or error messages.

Inputs

help
A hint to display, helping the user when filling a form field (optional)
errors
ValidationErrors of a formControl (optional)
showErrors
Activate the display of error messages (disable for example when formControl is pristine if you don't want to display errors at initial state) (default: true)
showAllErrors
Display all errors or only the first error encountered (errors being sorted by key) (default: true)
errorMessages
Provides default messages to display for the possible error keys when none is present in the errors object. If a message (string) is provided for a given key, it will be displayed rather than the message provided in errorMessages.
id
Id for the hint message, can be used for accessibility aria-describedby attribute in parent component
