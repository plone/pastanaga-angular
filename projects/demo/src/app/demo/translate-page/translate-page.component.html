Translate Pastanaga allows to manage translation.

Providing the translations

Each supported language is a JSON file where translated strings are handled in nested dictionaries:

                {{translations}}

Translations must be provided in the main app module:

                {{appModule}}

We can cascade several translations using the mergeTranslations method: 

            {{concatTranslations}}

Using the directive

<span translate>demo-page.title</span>

demo-page.title

With parameters

<span translate [translateParams]="{points: 10, total: 25}">demo-page.score</span>

demo-page.score

Using the pipe

{{ 'demo-page.title' | translate}}

Pipe with parameters

{{ 'demo-page.score' | translate:{points: 10, total: 25} }}

From a secondary translation file

<span translate>other.test</span>

other.test

Overrided by a secondary translation file

<span translate>common.yes</span>

common.yes
