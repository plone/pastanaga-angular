Inherited from BaseTextField

Inputs

debounceDuration
Duration (in milliseconds) used to debounce values emitted by valueChange (default: 500ms)
errorMessage
Error message to display when field validation is done externally. Will override internal validation error if any. (optional)
errorMessages
Map of all possible internal error messages (like required) (optional: by default field will have the error state without any message displayed)
placeholder
Field's placeholder (optional)
readonly
Set readonly state (default: false)
value
Field's value

Outputs

blur
Emit current value on blur
enter
Emit KeyboardEvent and string value when pressing enter
focus
Emit FocusEvent on focus
instantValueChange
Emit value as soon as it changes
keyUp
Emit string value on key up event
valueChange
Emit current value after a debounce time defined by debounceDuration input
